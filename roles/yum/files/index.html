Nightking CA Certificate link

Click this link for the Nightking CA certificate. Install it using one of the below methods and rewrite the URL of the website to HTTPS afterwards to get access to the Grafana web server.

Mozilla Firefox

Firefox will automatically ask you if you want to trust the new Certificate Authority. Check the box next to "Trust this CA to identify websites" and click OK.

Google Chrome / Safari / other

Double-click on the downloaded certificate and the OS will ask you to install the certificate to the trusted root store. (Tested on Mac.)

Apple OSX Console

You can install the certificate programmatically. This affects browsers that use the operating system keychain, like Safari. Open the console and run:

cd Downloads

sudo security add-trusted-cert -d -r trustRoot -k "/Library/Keychains/System.keychain" "nightkingca.crt"
